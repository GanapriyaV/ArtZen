🎨 ArtZen - Android Art Gallery App ArtZen is a simple and elegant Android application built using Jetpack Compose that showcases beautiful nature artworks. Users can browse through images using navigation buttons and view artwork details like title and author.

🚀 Features 📸 Displays curated nature images ⏮️ Previous & Next navigation buttons 🖼️ Image with title and author details 🎨 Clean and responsive UI using Jetpack Compose 🔄 Dynamic state handling using remember

🛠️ Tech Stack Language: Kotlin UI: Jetpack Compose IDE: Android Studio

This project is built as part of Android development learning and practice.
